Minecraft介绍

在 Minecraft 世界中体验各式各样的探险、生存与建造乐趣，无论是 Minecraft：电脑版豪华合集、Java 版和 Bedrock 版还是 Minecraft Launcher 均可畅玩！在电脑上畅玩 Bedrock 版，在 Minecraft 商城里浏览无穷无尽的精彩社区内容，在不同的地图中探索新玩法，使用角色创建道具和表情尽情表达自我。*

*Minecraft：Java 版可在 Windows、MAC 和 Linux 上运行，Minecraft：Bedrock 版可在 Windows 上运行。豪华合集仅能在适用于 Windows 的 Minecraft：Bedrock 版上运行。

保持足智多谋

投身充满无限可能的冒险世界，立志成为令人称羡的能工巧匠。悉心观察、深入探索周围的广袤环境，从郁郁葱葱的森林采伐坚实木材，于嶙峋山脉挖掘坚硬矿石，自潺潺溪流收集圆润石子，让这些素材成为搭建梦想建筑的基石 。

挺过黑夜

在这片神秘未知的领域，务必要时刻保持警觉，始终与那些四处游走的生物维持安全距离，以此规避难以预料的危险。这些生物习性诡异，行为难以捉摸，一旦靠得太近，从未有人知晓会引发怎样可怕的后果，也许是致命攻击，也许是被拖入未知险境，所以远离它们是明智之举 。

建造令人惊奇的作品

踏入充满惊喜的红石世界，悉心钻研，全力探寻红石粉的多元使用方法。借助红石粉独特的能量传导特性，巧妙搭建复杂电路，赋予创造作品灵动生命力，使其栩栩如生。或是构建奇妙机关，让作品在一众创作中脱颖而出，瞬间吸引众人目光，成为令人瞩目的焦点。

探索 Minecraft

在这款终极沙盒游戏中建造出你能想到的所有奇景，揭开诡异秘事的神秘面纱， 并在漫漫长夜中生存下来。在 Minecraft 中，每次游玩都有不同体验，世界各处都蕴藏着让人难以忘怀的冒险。 在无限世界中探索创造，一步一个方块，这个世界任你塑造。
